📊 Superstore Sales — End-to-End Data Analysis Project

🚀 Overview

This project analyzes an e-commerce Superstore dataset to uncover business insights, trends, and performance metrics. It demonstrates the complete data analysis pipeline from data cleaning to visualization and insights.

---

🎯 Objectives

* Analyze sales, profit, and discount patterns
* Identify loss-making areas
* Understand customer and regional performance
* Provide actionable business insights

---

🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

📁 Project Workflow

1. Data Generation / Collection
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Business Insights

---

📊 Key Analysis Performed

* Category-wise sales and profit analysis
* Sub-category profit margin analysis
* Discount vs Profit relationship
* Regional performance comparison
* Monthly sales trends
* Correlation analysis

---

📈 Key Insights

* High discounts (≥30%) lead to losses
* Some sub-categories consistently generate negative profit
* Certain regions outperform others in revenue and margin
* Seasonal trends observed in monthly sales

---

📷 Sample Outputs

(All charts are saved in the `superstore_outputs/` folder)

* Sales vs Profit by Category
* Discount vs Profit Curve
* Monthly Sales Trend
* Correlation Heatmap
